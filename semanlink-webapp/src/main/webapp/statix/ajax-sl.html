Access to Semanlink output from an HTML page

This page shows one simple way to display Semanlink data from an HTML page using Ajax.

This page is a static HTML page (it is not generated by a servlet): you can access Semanlink data from within your application, just by including some javascript. You can save a copy of it on your computer and open it locally and it should work.
